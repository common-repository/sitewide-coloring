== Description ==
Plugin that allows show banners on all pages or posts just in few clicks without coding and changing post.php or page.php. Upon activation of the plugin user is able to configure:

### Features of Sitewide Coloring plugin
*   unlimited number of banners
*   banners for desktop and mobile versions
*   positioning of banners (below, above, on the right/left)
*   per-paragraph placement of banners
*   whether coloring will be shown only on pages, only on posts or both
*   exclusions and pages/posts where banners will not be shown

Plugin also includes a small preview of current configuration.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings-> screen to configure the plugin

== Frequently Asked Questions ==

= What kind of banners should be added to plugin? =
Any! You simply paste html code of your banner. These may be AdSense codes, images uploaded to Media or banners with links.

= Does the plugin support banners in shortcodes? =
Of course, it does. We recommend first creating banners in Datafeedr plugin, getting the shortcode there and then inserting it in Sitewide coloring plugin settings.

= Could the plugin enable different banners on posts and pages? =
No, this option is not supported right now. However, you may enable banners only on posts or only on pages.

= Can I disable showing ads on pages with specific parent? =
Yes, this option is currently available in plugin settings.
